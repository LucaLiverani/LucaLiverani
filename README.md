# Hi, I'm Luca

I work with data: I model them, I build the pipelines that move them, I visualize them, and I'm also playing with the AI layer on top.

I decided to make most of my repositories private — LLMs and crawlers scrape the public web every minute, and I'd rather not have my code quietly pulled into someone's training set.
It's not really about secrecy — I'd just rather decide for myself what I put out there.

One project is public, though, and it's probably the most personal thing I've built.

## [My Life in Data](https://mylife-in-data.com)

[![My Life in Data dashboard](https://raw.githubusercontent.com/LucaLiverani/mylife-in-data/5a288406803bea14a7f080846c81847a4d21a61d/docs/screenshots/home.png)](https://mylife-in-data.com)

A personal data platform that pulls my own activity back out of Spotify, YouTube, Google Maps, and Google Calendar, loads it into a warehouse, and turns it into a dashboard. Spotify and Google already keep a detailed record of everything I do. This is my version of that record, except I own it and you're allowed to look.

It's a full end-to-end build: streaming ingestion, a ClickHouse warehouse, dbt models, Dagster orchestration, and a React frontend on Cloudflare. The code and the docs are all in the repo.

- Live: https://mylife-in-data.com
- Code: https://github.com/LucaLiverani/mylife-in-data
